LoRa Wireless Communication Protocol
Trail Capacity Monitoring System

Using a low-power, long range wireless communications protocol, we have developed a trail capacity monitoring system. This system collects a count from IoT sensors and forwards that count to a webpage with an embedded Google Map. The numbers are displayed on the Google Map as heat to depict how many people are on a given section of trail. Click through the slides to learn more about the components.

BYU Capstone Project

This picture demonstrates what this will look like when deployed. A LoRa network gateway will connect arudino nodes over the LoRa protocol. (915MHz in the United States)

This is a LoRa gateway. This device captures frames sent from arduino nodes and forwards them over a backhaul network. In our case using fiber optic cabling. Could be implemented with cellular networks, or WiFi.

The arduino board used to work with the motion detection sensor and the LoRa protocol. We developed an alogrithm that would accurately count the number of users that passed by the sensors. This device uses extremely low power, and runs completely off solar power.

The microwave sensor that we used to detect people as they walk by. Connects to the arduino board using GPIO pins.
